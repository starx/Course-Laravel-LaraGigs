## Course notes

### Helper functions

#### `dd`

Short for `die and dump`. Ends the execution and show what `$var` (in the example below) has.

Usages:

    dd($var);

#### `ddd`

Short for `die, dump and debug`. Show a lot debug information at the point where this helper method was called.

Usages:

    ddd($var);
